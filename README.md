# Sinhala & Tamil New Year 2026 🎉

A React-based greeting website celebrating **Sinhala & Tamil New Year **.  
This project displays a festive New Year greeting with traditional design elements.

---

## 🌸 Project Description

This project was built using **React** to create a modern greeting page for Sinhala & Tamil New Year 2026.  
It includes decorative components, cultural elements, and a festive message to celebrate the occasion.

---

## 🚀 Getting Started

This project was bootstrapped with Create React App.

### Prerequisites

Make sure you have installed:

- Node.js
- npm (comes with Node.js)
